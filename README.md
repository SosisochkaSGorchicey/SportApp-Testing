# SportApp-Testing
A sport app showing today finished basketball matches in the world.  
Used libraries: 
- Jetpack Compose  
- Retrofit  
- OkHttp
- Coil
- Coroutines
Also has connection with OneSignal and sport api - https://allsportsapi.com/basketball-api
